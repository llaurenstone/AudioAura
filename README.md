# AudioAura

AudioAura is a music analytics web application inspired by Spotify Wrapped.
Users log in with Spotify and receive a visual report of their listening habits,
including mood, energy, tempo, and genre trends.

## Tech Stack (Planned)
- Frontend: React + Vite
- Backend: Node.js + Express
- API: Spotify Web API
- Auth: Spotify OAuth
- Visualization: Chart.js / D3
